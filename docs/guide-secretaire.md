## 1. Tableau de bord

Votre tableau de bord centralise les informations urgentes du bureau :
- Dossiers en attente de traitement
- Rendez-vous du jour
- Courriers non traités
- Messages non lus

## 2. Gestion des étudiants

Dans **Étudiants**, vous accédez à la liste complète de tous les étudiants enregistrés sur la plateforme.

### Rechercher un étudiant

- Saisissez un nom ou une adresse e-mail dans la barre de recherche
- Filtrez par statut de dossier
- Cliquez sur un étudiant pour accéder à sa fiche

### Informations en liste

| Colonne | Description |
|---|---|
| Photo / Nom | Identité de l'étudiant |
| E-mail | Adresse e-mail de contact |
| Statut | État du dossier (nouveau, en cours, soumis, validé) |
| Progression | Pourcentage d'avancement du dossier |
| Date | Date d'inscription |

---

## 3. Fiche détail d'un étudiant

Identique à celle du conseiller, la fiche détail vous donne accès à toutes les informations d'un étudiant :

- **Profil** : nom, e-mail, téléphone, photo
- **Checklist** : 7 étapes d'avancement du dossier
- **Candidatures** : écoles et formations choisies, avec notes et statuts
- **Documents** : liste des pièces téléversées, avec téléchargement
- **Documents officiels à transmettre** : prise en charge, AEVM, hébergement, salaire, carte de séjour
- **Projet d'études** : pays, niveau, formation, présentation

### Accès rapide à la messagerie

Le bouton **Message** en haut de la fiche ouvre directement la conversation avec l'étudiant.

## 4. Validation des dossiers et documents

Dans **Validations**, vous pouvez examiner et traiter les dossiers et documents en attente.

### Onglet Dossiers

Affiche les dossiers soumis par les étudiants en attente de validation. Pour chaque dossier :
1. Consultez les candidatures incluses
2. Cochez **Frais d'inscription reçus** si applicable
3. Cliquez sur **Valider** ou **Refuser**

### Onglet Documents

Affiche les documents individuels à examiner :
1. Téléchargez le document pour le vérifier
2. Ajoutez une note si nécessaire
3. Cliquez sur **Valider** ou **Rejeter**

### Archives

Basculez sur **Traités** pour consulter l'historique des validations.

## 5. Gestion du courrier

Dans **Courriers**, gérez tout le courrier entrant et sortant de Rézo Campus.

### Enregistrer un courrier

1. Cliquez sur **Nouveau courrier**
2. Renseignez les informations :
   - **Titre** : objet du courrier
   - **Type** : Entrant ou Sortant
   - **Expéditeur / Destinataire**
   - **Date** du courrier
   - **Notes** : informations complémentaires
3. Téléversez le scan ou le fichier numérique si disponible
4. Enregistrez

### Consulter le courrier

La liste affiche tous les courriers enregistrés, triés par date. Cliquez sur un courrier pour en voir le détail et télécharger le fichier associé.

### Filtres

- Filtrez par type (entrant / sortant)
- Recherchez par titre, expéditeur ou destinataire
- Filtrez par période

## 6. Gestion des clients

Dans **Clients**, gérez la base de données des clients de Rézo Campus (particuliers et entreprises).

### Types de clients

| Type | Utilisation |
|---|---|
| Individuel | Particulier (prénom, nom) |
| Entreprise | Société partenaire ou cliente |

### Créer un client

1. Cliquez sur **Nouveau client**
2. Sélectionnez le type (individuel ou entreprise)
3. Renseignez les informations de contact :
   - Prénom / Nom (ou nom d'entreprise)
   - E-mail
   - Téléphone
   - Adresse physique
4. Ajoutez des notes si nécessaire
5. Enregistrez

### Modifier ou supprimer un client

Depuis la fiche client, modifiez les informations ou supprimez le client. La suppression est irréversible.

## 7. Rendez-vous clients

Dans **Rendez-vous clients**, planifiez et suivez les rendez-vous avec les clients de Rézo Campus.

### Planifier un rendez-vous

1. Cliquez sur **Nouveau rendez-vous**
2. Sélectionnez le client dans la liste
3. Définissez la date, l'heure et la durée
4. Renseignez le lieu et l'ordre du jour
5. Enregistrez

### Suivi post-rendez-vous

Après le rendez-vous, renseignez le compte-rendu :
- **Le RDV a-t-il eu lieu ?** (Oui / Non)
- **Sujets discutés**
- **Résolutions prises**
- **Perspectives** (prochaines étapes)
- **Motif d'échec** (si le RDV n'a pas eu lieu)

### Statuts des rendez-vous

| Statut | Description |
|---|---|
| Planifié | RDV à venir |
| Terminé | RDV passé avec compte-rendu |
| Annulé | RDV annulé |

## 9. La messagerie

Dans **Messagerie**, communiquez avec les étudiants et les membres de l'équipe interne.

- Sélectionnez un contact dans la liste
- Envoyez vos messages et pièces jointes
- Les messages non lus apparaissent en gras

## 10. Facturation

Dans **Facturation**, consultez les factures créées par la comptabilité.

> En tant que secrétaire, vous avez accès à la consultation et à la création de factures. Pour toute modification, coordonnez-vous avec la comptabilité.

### Consulter une facture

1. Cliquez sur une facture dans la liste
2. Visualisez les lignes de facturation, le total et le statut

### Créer une facture

1. Cliquez sur **Nouvelle facture**
2. Sélectionnez le client
3. Ajoutez les lignes de service (description + montant)
4. Définissez la date et les conditions
5. Enregistrez en brouillon ou envoyez directement

### Statuts de facturation

| Statut | Description |
|---|---|
| Brouillon | Facture en cours de rédaction |
| Envoyée | Facture transmise au client |
| Payée | Paiement confirmé |
| Annulée | Facture annulée |

---

## Bonnes pratiques

- **Enregistrez chaque courrier** dès réception ou envoi pour assurer la traçabilité
- **Complétez les comptes-rendus** de rendez-vous dans les 24h suivant le RDV
- **Maintenez la base clients** à jour (coordonnées, notes)
- **Traitez les dossiers en attente** en priorité pour ne pas bloquer l'avancement des étudiants
