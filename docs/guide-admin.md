## 2. Gestion des utilisateurs

Dans **Utilisateurs**, créez et gérez tous les comptes de la plateforme.

### Rôles disponibles

| Rôle | Description |
|---|---|
| Admin | Accès complet à toutes les fonctionnalités |
| Conseiller | Suivi des étudiants, validations, messagerie |
| Secrétaire | Administration, courriers, clients |
| Comptable | Comptabilité, facturation, transactions |
| Chef de projet | Gestion de projets, cahiers des charges |
| Commercial | Suivi commercial, clients, activités |
| RH | Personnel, réunions, planification |
| École | Espace établissement partenaire |
| Étudiant | Espace personnel étudiant |

### Créer un compte utilisateur

1. Cliquez sur **Nouvel utilisateur**
2. Renseignez :
   - Nom complet
   - Adresse e-mail (sera utilisée pour la connexion)
   - Rôle(s) attribué(s)
   - Pour le rôle *École* : sélectionnez l'établissement lié
3. L'utilisateur reçoit un e-mail d'invitation pour définir son mot de passe

### Modifier un compte

Depuis la fiche utilisateur, vous pouvez :
- Modifier le nom, l'e-mail, le rôle
- Bloquer un compte (l'utilisateur ne peut plus se connecter)
- Réinitialiser le mot de passe
- Supprimer le compte

### Bloquer un compte

Le blocage est réversible. Un compte bloqué garde ses données mais ne peut plus accéder à la plateforme. Utile en cas de départ d'un collaborateur ou d'un problème avec un établissement partenaire.

> La suppression d'un compte est irréversible et entraîne la perte de l'historique lié à cet utilisateur.

### Rattachement d'un utilisateur École

Lors de la création d'un compte avec le rôle *École*, sélectionnez l'établissement correspondant dans la liste **École rattachée**. Cet utilisateur n'aura accès qu'aux candidatures de son établissement.

## 3. Supervision des dossiers étudiants

Dans **Dossiers**, accédez à l'ensemble des dossiers étudiants de la plateforme.

### Vue d'ensemble

La liste affiche tous les étudiants avec :
- Leur profil (photo, nom, e-mail)
- Le statut du dossier
- Le pourcentage d'avancement
- Les candidatures en cours

### Fiche détail

Cliquez sur un étudiant pour ouvrir sa fiche complète (identique à la vue conseiller) :
- Checklist d'avancement en 7 étapes
- Candidatures avec notes conseiller et frais d'inscription
- Documents téléversés
- Documents officiels envoyés
- Projet d'études

### Actions disponibles

- Télécharger les documents
- Envoyer des documents officiels (prise en charge, AEVM, hébergement, etc.)
- Accéder à la messagerie de l'étudiant

## 4. Validation des dossiers et documents

Dans **Validations**, supervisez et traitez les éléments en attente.

### Onglet Dossiers

Validez ou refusez les dossiers de candidature soumis par les étudiants. Avant validation :
- Vérifiez que les documents requis sont présents
- Cochez **Frais d'inscription reçus** si applicable
- Ajoutez une note si nécessaire

### Onglet Documents

Examinez les documents individuels :
1. Téléchargez et vérifiez chaque document
2. Ajoutez une note pour l'étudiant (explique le motif d'un rejet)
3. Validez ou rejetez

### Notification de signature post-validation école

Quand un établissement valide une candidature, vous recevez une notification : **"Veuillez faire signer le document de validation chez Rézo Campus"**. Cette notification demande à l'équipe de collecter la signature officielle de Rézo Campus sur l'attestation.

## 5. Gestion des écoles partenaires

Dans **Écoles**, gérez le catalogue des établissements partenaires.

### Créer un établissement

1. Cliquez sur **Nouvel établissement**
2. Renseignez :
   - Nom de l'établissement
   - Ville et pays
   - Site web
   - Logo (image PNG ou JPG)
3. Enregistrez

### Gérer les formations

Pour chaque établissement, cliquez sur **Formations** pour ajouter, modifier ou supprimer des formations :
- Nom de la formation
- Niveau (Licence, Master, BTS, etc.)
- Domaine (Informatique, Commerce, Santé, etc.)
- Langue d'enseignement
- Durée

### Lier un utilisateur à un établissement

Lors de la création d'un compte *École* (voir section 2), sélectionnez l'établissement. Vous pouvez également modifier ce rattachement depuis la fiche utilisateur.

## 9. Facturation

Dans **Facturation**, supervisez et gérez toutes les factures de Rézo Campus.

### Créer une facture

1. Cliquez sur **Nouvelle facture**
2. Sélectionnez le client
3. Ajoutez les lignes de service :
   - Description du service
   - Montant (en FCFA ou devises)
4. Définissez la date d'émission et les conditions de paiement
5. Enregistrez en brouillon ou envoyez directement

### Gérer les factures

Pour chaque facture, vous pouvez :
- Modifier les lignes (si en brouillon)
- Changer le statut (brouillon → envoyée → payée)
- Annuler une facture
- Télécharger / Imprimer

### Tableau de bord financier

Le tableau de bord comptable affiche :
- Total facturé sur la période
- Factures en attente de paiement
- Factures payées
- Factures en retard

### Statuts

| Statut | Description |
|---|---|
| Brouillon | En cours de rédaction, non envoyée |
| Envoyée | Transmise au client |
| Payée | Paiement confirmé |
| Annulée | Annulée (gardée pour archive) |

## 11. Droits et responsabilités

### Ce que seul l'administrateur peut faire

- Créer, modifier, bloquer ou supprimer des comptes utilisateurs
- Créer et modifier les établissements partenaires et leurs formations
- Accéder à toutes les données sans restriction

### Bonnes pratiques de sécurité

- **Ne partagez jamais vos identifiants** admin — chaque collaborateur doit avoir son propre compte
- **Bloquez immédiatement** le compte d'un collaborateur qui quitte Rézo Campus
- **Vérifiez régulièrement** les comptes inactifs et bloquez-les si nécessaire
- **Utilisez des mots de passe forts** pour les comptes admin (16 caractères minimum)
- **N'attribuez le rôle Admin** qu'aux personnes qui en ont réellement besoin

### Gestion des incidents

En cas de problème de sécurité (compte compromis, accès non autorisé) :
1. Bloquez immédiatement le compte concerné
2. Réinitialisez le mot de passe
3. Vérifiez les actions récentes dans les logs (contacter le support technique)
