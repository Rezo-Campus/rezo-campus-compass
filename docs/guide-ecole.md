## 1. Accéder à votre espace établissement

Votre compte vous est créé par l'équipe Rézo Campus et lié à votre établissement. Connectez-vous avec les identifiants fournis. Votre accès est limité aux candidatures et étudiants rattachés à votre établissement — vous ne voyez pas les dossiers des autres établissements.

- Nombre de candidatures reçues

## 3. Gérer vos formations

Dans **Formations**, consultez et gérez les formations rattachées à votre établissement.

### Informations disponibles par formation

| Champ | Description |
|---|---|
| Nom de la formation | Intitulé complet |
| Niveau | Licence, Master, BTS, etc. |
| Domaine | Informatique, Commerce, Santé, etc. |
| Langue d'enseignement | Français, Anglais, etc. |
| Durée | Durée de la formation |

> Pour ajouter ou modifier une formation, contactez l'équipe Rézo Campus (Admin).

## 4. Examiner les candidatures

Dans **Candidatures**, vous visualisez tous les dossiers étudiants transmis à votre établissement par Rézo Campus.

### Informations disponibles par dossier

**Profil de l'étudiant**
- Nom complet, e-mail, téléphone
- Photo de profil (si renseignée)

**Formation visée**
- Nom de la formation, niveau, domaine

**Documents joints** (avec statut de validation par Rézo Campus)
- Pièce d'identité / Passeport
- Diplôme
- Relevé de notes
- Lettre de motivation
- CV
- Autres documents

Chaque document est accompagné de son statut : *Validé* (vert), *En attente* (jaune) ou *Rejeté* (rouge).

> Les documents marqués **Requis** (diplôme et pièce d'identité) sont indispensables pour valider une candidature.

### Note du conseiller

Si le conseiller Rézo Campus a laissé une note à destination de votre établissement, elle apparaît dans un encadré ambré en haut du dossier. Cette note peut mentionner :
- Un document en cours de renouvellement
- Une information particulière sur l'étudiant
- Toute autre précision utile pour votre examen

### Statut de la candidature

Vous pouvez modifier le statut d'une candidature via le menu déroulant :

| Statut | Signification |
|---|---|
| En cours de traitement | Dossier reçu, examen en cours |
| Pré-sélectionné | Étudiant sélectionné pour la suite |
| Entretien planifié | Entretien à venir avec l'étudiant |
| Accepté | Admission confirmée |
| Refusé | Candidature non retenue |

## 5. Valider une candidature

La validation officielle est distincte du statut de candidature. Elle génère l'**attestation de validation** que l'étudiant pourra télécharger.

### Prérequis obligatoires

Avant de pouvoir valider, ces deux documents doivent être présents dans le dossier :
- **Diplôme** (ou relevé de notes)
- **Pièce d'identité** (passeport ou CNI)

Si l'un de ces documents est absent, un message d'avertissement s'affiche.

### Procédure de validation

1. Ouvrez le dossier de l'étudiant dans **Candidatures**
2. Vérifiez la présence des documents requis
3. Cliquez sur le bouton **Valider la candidature** (distinct du menu de statut)
4. Confirmez dans la boîte de dialogue

**Effets de la validation :**
- L'étudiant reçoit une notification de félicitations
- Les conseillers et admins Rézo Campus reçoivent une notification pour faire signer le document
- Un lien **Attestation** apparaît dans le dossier
- L'étudiant peut désormais accéder à son attestation depuis son espace

## 6. Téléverser le cachet et la signature

Une fois la candidature validée, le lien **Attestation** apparaît dans le dossier. Accédez à la page d'attestation pour y apposer le cachet officiel de votre établissement.

### Procédure

1. Cliquez sur **Attestation** dans le dossier validé
2. Sur la page d'attestation, descendez jusqu'à la section **Cachet et Signature**
3. Cliquez sur **Téléverser le cachet / la signature**
4. Choisissez une image PNG ou JPG (fond transparent recommandé)
5. Le cachet est immédiatement affiché sur l'attestation

> Le cachet apparaît dans la zone prévue à cet effet sur le document A4. L'étudiant verra automatiquement la version mise à jour.

## 7. Transmettre des documents à l'étudiant

Sur la page d'attestation de chaque candidature validée, vous pouvez transmettre des documents administratifs directement à l'étudiant.

### Types de documents disponibles

| Type | Description |
|---|---|
| Convocation | Convocation à la rentrée ou à un entretien |
| Lettre d'admission | Lettre officielle d'admission |
| Programme de cours | Programme de la formation |
| Attestation d'inscription | Attestation d'inscription définitive |
| Autre document | Tout autre document utile |

### Procédure d'envoi

1. Sur la page d'attestation, repérez la section **Documents administratifs pour l'étudiant**
2. Sélectionnez le type de document dans la liste déroulante
3. Cliquez sur **Téléverser**
4. Choisissez le fichier sur votre ordinateur
5. Le document est envoyé et l'étudiant reçoit une notification

L'étudiant peut télécharger les documents transmis depuis son espace **Docs officiels**.

## 8. La messagerie

Dans **Messages**, communiquez directement avec vos candidats.

- La liste de contacts affiche automatiquement les étudiants qui ont postulé à votre établissement
- Cliquez sur un étudiant pour ouvrir la conversation
- Les échanges sont privés entre votre établissement et l'étudiant concerné

> Utilisez la messagerie pour informer l'étudiant d'une convocation, d'un entretien ou de toute demande complémentaire.

## Questions fréquentes

**Je ne vois pas certains documents dans le dossier.**
Seuls les documents téléversés et validés par Rézo Campus sont visibles. Si un document important manque, signalez-le à votre contact Rézo Campus.

**La validation est bloquée — il me demande des documents manquants.**
Le diplôme ET la pièce d'identité sont obligatoires. Si l'un des deux n'est pas dans le dossier, vous ne pouvez pas valider. Informez Rézo Campus pour qu'ils complètent le dossier.

**Comment modifier une validation déjà effectuée ?**
Une validation est définitive. En cas d'erreur, contactez l'équipe Rézo Campus.

**Un étudiant me dit qu'il ne voit pas son attestation.**
Vérifiez que la candidature affiche bien la date de validation (encadré vert). Si oui, demandez à l'étudiant de se connecter et d'aller dans son menu **Docs officiels**.
