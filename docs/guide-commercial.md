## 2. Activités commerciales

Dans **Activités**, enregistrez et suivez vos actions commerciales : appels, prospections, relances, présentations, etc.

### Enregistrer une activité

1. Cliquez sur **Nouvelle activité**
2. Renseignez :
   - **Type d'activité** : appel, e-mail, réunion, visite, etc.
   - **Client ou prospect concerné**
   - **Date et heure**
   - **Résumé** de l'activité
   - **Résultat** : positif, négatif, en attente
   - **Prochaine action** planifiée
3. Enregistrez

### Consulter les activités

La liste des activités peut être filtrée par :
- Client
- Type
- Période
- Résultat

## 3. Rendez-vous clients

Dans **RDV Clients**, gérez votre agenda de rendez-vous avec les clients et prospects de Rézo Campus.

### Planifier un rendez-vous

1. Cliquez sur **Nouveau rendez-vous**
2. Sélectionnez le client dans la liste (ou créez-en un nouveau via la Secrétaire)
3. Définissez :
   - Date et heure
   - Durée
   - Lieu (bureau, en ligne, chez le client)
   - Ordre du jour
4. Enregistrez

### Après le rendez-vous

Renseignez le compte-rendu pour garder une trace commerciale :
- **Le RDV a-t-il eu lieu ?**
- **Sujets discutés** : besoins du client, objections, intérêts
- **Résolutions** : décisions prises, engagements
- **Perspectives** : suite à donner, prochaine étape
- **Motif d'échec** : si le RDV n'a pas pu avoir lieu

### Bonnes pratiques

- Complétez le compte-rendu **dans l'heure** suivant le rendez-vous, quand les détails sont frais
- Planifiez **systématiquement** la prochaine action avant de clore un RDV

## 4. Facturation

Dans **Facturation**, créez les devis et factures liés à vos activités commerciales.

### Créer une facture

1. Cliquez sur **Nouvelle facture**
2. Sélectionnez le client
3. Ajoutez les lignes de service
4. Enregistrez en brouillon

> La validation finale et le suivi des paiements relèvent de la comptabilité. Transmettez vos brouillons à la comptable pour traitement.

### Consulter les factures

Vous avez accès à toutes les factures de Rézo Campus. Filtrez par client ou par statut pour retrouver rapidement une facture.

## Bonnes pratiques commerciales

- **Documentez chaque interaction** client dans les activités pour maintenir un historique fiable
- **Renseignez les comptes-rendus** de RDV pour permettre à l'équipe de reprendre le dossier si nécessaire
- **Coordonnez-vous avec la secrétaire** pour la création des fiches clients et la planification des RDV partagés
- **Transmettez vos factures** à la comptabilité dès qu'un accord client est conclu
