## 2. Facturation

Dans **Facturation**, créez, gérez et archivez toutes les factures de Rézo Campus.

### Créer une facture

1. Cliquez sur **Nouvelle facture**
2. Sélectionnez le **client** dans la liste déroulante
3. Définissez la **date d'émission**
4. Ajoutez les **lignes de facturation** :
   - Description du service rendu
   - Montant unitaire
   - Utilisez le bouton **Ajouter une ligne** pour chaque prestation supplémentaire
5. Renseignez les **conditions de paiement** (délai, mode de paiement)
6. Ajoutez des **notes** si nécessaire
7. **Enregistrez en brouillon** pour révision ultérieure, ou **Envoyez** directement

### Détail d'une facture

Cliquez sur une facture dans la liste pour accéder à :
- Le récapitulatif complet (client, lignes, total, TVA si applicable)
- L'historique des statuts
- Les actions disponibles

### Gérer le cycle de vie d'une facture

| Action | Quand l'effectuer |
|---|---|
| Passer en "Envoyée" | Dès transmission au client |
| Passer en "Payée" | Dès réception du paiement |
| Annuler | En cas d'erreur ou d'accord client |

> Une facture payée ne peut pas être modifiée. Créez un avoir si une correction est nécessaire.

### Statuts de facturation

| Statut | Couleur | Description |
|---|---|---|
| Brouillon | Gris | En cours de rédaction |
| Envoyée | Bleu | Transmise au client |
| Payée | Vert | Paiement confirmé |
| Annulée | Rouge | Annulée définitivement |

### Numérotation des factures

Chaque facture reçoit un numéro unique défini lors de la création. Respectez la convention de numérotation établie par Rézo Campus (ex. : RC-2026-001).

## 3. Transactions

Dans **Transactions**, enregistrez et consultez les entrées et sorties financières de Rézo Campus.

### Enregistrer une transaction

1. Cliquez sur **Nouvelle transaction**
2. Sélectionnez le type :
   - **Entrée** : paiement reçu, frais d'inscription, prestation encaissée
   - **Sortie** : charge, achat, remboursement
3. Renseignez :
   - Montant
   - Date
   - Description / libellé
   - Référence (numéro de facture, reçu, etc.)
4. Enregistrez

### Vue des transactions

La liste des transactions peut être filtrée par :
- Période (semaine, mois, trimestre, année)
- Type (entrées / sorties)
- Montant

## Bonnes pratiques comptables

- **Enregistrez chaque paiement** dès réception, avec la référence de la facture correspondante
- **Faites des rapprochements** réguliers entre les transactions et les factures payées
- **Archivez** toutes les pièces justificatives numériques dans le système
- **Clôturez le mois** en vérifiant que toutes les transactions sont enregistrées avant le 5 du mois suivant
- **Signalez** immédiatement les factures impayées dépassant leur échéance à l'administration
